# Accesibilidad en Adobe Admin Console

Se aplica a empresas y equipos.

Obtenga información acerca de la compatibilidad con la accesibilidad en Adobe Admin Console, que proporciona una ubicación central para administrar las autorizaciones de Adobe en toda la organización.

## Compatibilidad de accesibilidad del sistema operativo

Las funciones de accesibilidad de muchos sistemas operativos ampliamente utilizados se pueden utilizar en Admin Console. Estas funciones de accesibilidad incluyen alto contraste y otros ajustes de visualización. Sin embargo, puede haber algunas limitaciones debido a la interfaz de usuario actual de Admin Console.

## Compatibilidad con teclado

La mayor parte del contenido interactivo de Admin Console se puede navegar y utilizar sólo con el teclado. Por ejemplo, puedes usar **[!UICONTROL Tab]** o **[!UICONTROL Shift]** + **[!UICONTROL Tab]** para la navegación y el control. Una vez que navegue al vínculo o botón correcto, presione la tecla **[!UICONTROL Intro]** o **[!UICONTROL Retorno]** para seleccionarlo. Puede que aún haya algunas excepciones como se describe en el [informe de conformidad con la accesibilidad](https://www.adobe.com/accessibility/compliance.html) (_solo en inglés_).

## Soporte de tecnología de asistencia

Admin Console ofrece cierta compatibilidad con tecnologías de asistencia (que incluyen lectores de pantalla y ampliadores de pantalla). Para obtener más información, vea [Informe de conformidad con la accesibilidad](https://www.adobe.com/accessibility/compliance.html).

## ¿Qué funciones de accesibilidad se admiten en Admin Console?

[Adobe Admin Console](https://adminconsole.adobe.com/) proporciona una ubicación central para administrar las autorizaciones de Adobe en toda la organización. Ofrece soporte de tecnología de asistencia, soporte de teclado y soporte de accesibilidad del sistema operativo (con algunas excepciones).

## ¿Tiene comentarios?

El equipo de Admin Console trabaja continuamente para mejorar su accesibilidad y la experiencia del usuario. Puede compartir sus comentarios sobre las características de accesibilidad de la [comunidad empresarial y de equipos](https://community.adobe.com/enterprise-teams-622).
