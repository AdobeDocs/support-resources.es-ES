# Información general de soporte de Adobe Workfront

Adobe Workfront se está uniendo a Adobe Admin Console Platform.

En los próximos meses, se contactará con los clientes para coordinar una migración a Adobe Identity y Admin Console, una consola administrativa del sistema más segura y funcional. Estos son algunos beneficios para el próximo cambio:

* Autentique para Workfront y otros productos mediante una sola identidad de Adobe.
* Administre las autorizaciones de usuario por separado entre los entornos de producción y de zona protegida o de vista previa.
* Utilice varias configuraciones de SSO para los usuarios de Workfront.
* Administre los derechos de Workfront del mismo modo que otros productos de Adobe.

Para obtener una revisión detallada de Adobe Identity y Admin Console, puede hacer referencia a esta [página de soporte técnico](https://helpx.adobe.com/es/enterprise/admin-guide.html). También se puede comparar la consola actual de Workfront y el nuevo Adobe Admin Console [aquí](https://one.workfront.com/s/document-item?bundleId=the-new-workfront-experience&topicId=Content%2FAdministration_and_Setup%2FGet_started-WF_administration%2Factions-in-admin-console.htm&_LANG=enus).

## Detalles de migración del cliente

Se contactó con los clientes existentes para programar las migraciones a partir de julio de 2022. Nuestro equipo de migración ayuda a sus compañeros a guiar a los clientes a través del proceso, les asesoró sobre la configuración de la consola y les proporcionó los vínculos a la documentación necesaria para que el traslado sea lo más sencillo y sencillo posible.

Los nuevos clientes netos a partir de julio de 2022 se colocaron automáticamente en Adobe Admin Console y no requirieron más acciones.
